Política de Privacidade

Nós nos limitamos a coletar a menor quantidade de dados do usuário de que precisamos a fim de criar uma experiência perfeita ao usar VideoConsul.

Esses dados incluem principalmente: RTCICECandidates & RTCSessionDescription que é necessário para estabelecer a chamada de vídeo usando WebRTC. RTCICECandidates, RTCSessionDescription não é armazenado em nenhum banco de dados persistente, é removido do servidor assim que o usuário sai da chamada fechando o navegador janela. Os fluxos de mídia são criptografados usando o transporte seguro em tempo real Protocolo (SRTP).

Para mensagens de bate-papo e compartilhamento de arquivos, usamos WebRTCs RTCDataChannel para transferir dados diretamente de um ponto a outro. Com RTCDataChannel, todos os dados são protegidos com Segurança da camada de transporte de datagrama (DTLS). DTLS é um derivado de SSL, o que significa que seus dados serão tão seguros quanto usando qualquer conexão baseada em SSL padrão.

Se você gravar uma reunião, a gravação da reunião será temporariamente armazenada em Seu navegador Blob, em seguida, será baixado em seu PC / dispositivo móvel.

VideoConsul é puramente ponto a ponto, o que significa que o usuário vídeo, áudio, chat e arquivo não são enviados para o nosso servidor.

Usamos o Google Analytics para rastrear estatísticas de uso agregadas a fim de melhorar nosso serviço. O criador de VideoConsul não tem intenção de usar pessoalmente ou vender qualquer um dos dados acima mencionados.
